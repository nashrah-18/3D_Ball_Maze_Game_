## OVERVIEW:
Basic 3D Game
A small game where you control a character or object. Move a ball through a simple maze.

## Instructions:
- The player uses arrow keys to move a ball in a 3D world
- A camera follows the ball, giving a third-person view
- Obstacles are placed to create a maze-like challenge
- A goal object (green box) is added, reaching it wins the game
- A simple UI text guides the player on what to do
